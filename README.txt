Timetables:Tallinn
==================

Tallinn public transport timetables in your Windows Phone.
(App For Windows Phone 7.5+ done in Visual C# with Silverlight)

Authors: Daniel Arruda & Steven Murekas
(Done in 24h during Tallinn WP7 Code Camp - 2012)

This software is released under FreeBSD license.

More info: https://github.com/dasanjos/TallinnTransport
